# HealthPulse

> 디지털 헬스케어 뉴스 수집 및 리포트 자동화 서비스

## 프로젝트 개요

디지털 헬스케어, 질병진단 키트 등을 생산하는 회사를 위한 **네이버 뉴스 기반 일일 리포트 자동화 시스템**입니다.

### 주요 기능

- **뉴스 자동 수집**: 네이버 뉴스에서 헬스케어 관련 기사 자동 수집
- **AI 기반 분석**: 기사 요약, 카테고리 분류, 중요도 산정
- **리포트 생성**: HTML/PDF 형태의 일일 리포트 자동 생성
- **이메일 발송**: 수신자 그룹별 맞춤 뉴스레터 자동 발송

## 시스템 아키텍처

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  뉴스 수집   │ -> │  데이터 처리  │ -> │  리포트 생성  │ -> │  이메일 발송  │
│  (Crawler)  │    │  (Filter/AI) │    │  (Template)  │    │   (SMTP)    │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
       ↑                                                        │
       └──────────────── 스케줄러 (매일 자동 실행) ─────────────────┘
```

## 기술 스택

| 구성요소 | 기술 | 비고 |
|---------|------|------|
| 언어 | Python 3.10+ | 크롤링/AI 라이브러리 풍부 |
| 스케줄러 | APScheduler / Airflow | 규모에 따라 선택 |
| DB | PostgreSQL / MongoDB | 뉴스 아카이빙용 |
| AI 요약 | OpenAI API / Claude API | 기사 요약 및 인사이트 |
| 이메일 | AWS SES / SendGrid | 대량 발송 시 |
| 배포 | AWS Lambda + CloudWatch | 서버리스로 비용 절감 |
